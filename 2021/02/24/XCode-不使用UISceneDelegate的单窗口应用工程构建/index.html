XCode:不使用UISceneDelegate的单窗口应用工程构建

Junfenghu
2021-02-24
XCode, iOS, 项目构建

  1. AppDelegate.h文件添加UIWindow属性，AppDelegate.m文件添加对应的视图控制器代码。

  2. self.window 的 rootViewController 的 backgroundColor 属性设置为任意颜色（默认是黑色）

  3. AppDelegate.m里 #pragma mark - UISceneSession lifecycle部分的函数代码删掉（这个很重要，不然界面还是一片黑色）

  4. Info.plist里 Application Scene Manifest 删掉

  5. SceneDelegate.h 与 SceneDelegate.m 文件删不删都可以
